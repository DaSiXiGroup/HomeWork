# UI Font Override

The sample does not bundle font files.

To replace raylib's default pixel font, put a TTF/OTF font here and name it:

- `ui.ttf`, or
- `ui.otf`

The application will try to load `assets/fonts/ui.ttf` first, then `assets/fonts/ui.otf`.

Recommended choices are fonts you already have permission to use, such as Noto Sans, MiSans, HarmonyOS Sans, Inter, or your system UI font.
